Platform

One workspace for ML experimentation and delivery.

Accelera is designed for teams that need repeatable pipelines, faster experiments, and a path from notebooks to services without rewriting the whole stack.

Graph pipelines

Create DAG-style workflows with preprocessing, model, prediction, metric, merge, and branch nodes.

Parallel execution

Run independent branches through a C++ backend and reuse fitted execution paths for inference.

Automated reports

Generate graph visualizations and HTML reports for model metrics, preprocessing, and comparisons.

Custom components

Plug in scikit-learn estimators or extend custom classifiers, regressors, clusterers, and transformers.

Modules

Built around the full ML pipeline lifecycle.

01

Accelera Pipe

02

Auto preprocessing

Prepare tabular, text, and image-classification datasets with reusable preprocessors and summaries.

Accelera AutoML

Benchmark platform

Workflow

Move from raw data to served predictions.

Accelera keeps experimentation explicit: every transformation and model branch is part of a graph, so results can be inspected, saved, and promoted into deployment artifacts.

  1. Prepare Load datasets and generate preprocessing pipelines for tabular, text, or image tasks.
  2. Experiment Build graph branches, run model comparisons, and capture metrics in reports.
  3. Optimize Use AutoML search and loop parallelization where runtime matters.
  4. Deploy Package fitted artifacts with schema validation and request tracking.

Designed for practical ML systems.

Use Accelera when your pipeline needs branching, reporting, native acceleration, and a clean transition from research code to reusable prediction services.

Supported Linux core features and code parallelizer
Partial Windows and macOS core workflows
Integrates scikit-learn, CMake, Clang, Graphviz, Docker

Quick Start

Install Accelera locally.

git clone https://github.com/Mohamed-Ashraf273/accelera.git
cd accelera
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pip install -e .
cmake -S . -B build
cmake --build build --parallel
