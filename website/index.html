Hybrid ML framework

Graph pipelines, AutoML, deployment, and native acceleration in one workspace.

Accelera combines a Python developer API with C++ execution paths for machine learning workflows. Build DAG-style pipelines, compare model branches, generate reports, automate preprocessing, search model spaces, package prediction services, and experiment with OpenMP loop parallelization.

What It Does

A practical framework for the full machine learning lifecycle.

The repository is more than a pipeline helper. It includes the core graph runtime, wrappers for reports and displays, automated preprocessing for several data types, AutoML search components, a model deployment module, benchmark web services, and a source-code parallelizer that can emit OpenMP-enabled C/C++.

Model graph engine

Build directed acyclic ML graphs using input, preprocess, model, predict, metric, merge, and branch nodes.

Optimized execution

Run independent branches through the C++ graph backend, share duplicated work, and reuse fitted execution paths.

Reports and visual output

Generate graph visualizations, metric displays, model reports, and preprocessing reports as HTML artifacts.

Service packaging

Package models, schemas, tracking logs, and Docker deployment flows for local, Heroku, and EC2 targets.

Architecture

Python APIs where you iterate, native paths where work gets heavy.

Accelera keeps the authoring interface in Python and delegates graph execution, loop analysis, and generated acceleration paths to native C/C++ components when the workflow benefits from lower-level control.

  1. Build the graph Compose preprocessing, models, predictions, metrics, branches, and merges using the Python API.
  2. Execute branches The C++ graph backend can schedule independent branches and reduce duplicate first-node work.
  3. Inspect outputs Reports and wrappers format metrics, arrays, plots, dictionaries, and pipeline diagrams for review.
  4. Promote artifacts Deployment commands snapshot model files, validate payload schemas, and serve predictions from containers.

Capabilities

Detailed functionality by area.

Core Pipeline

Branch multiple preprocessors and estimators, predict on test data, calculate supervised or unsupervised metrics, save/load pipelines, and generate graph reports.

Auto Preprocessing

Build reusable preprocessors for classical tabular datasets, text datasets, and image classification datasets, with wrappers for feature distributions and label summaries.

AutoML

Search classification and regression components, run SMAC optimization, warm-start from meta-features, and combine candidates with voting or stacked ensembles.

Deployment

Use a three-artifact model layout: preprocessing pipeline, feature selector, final model. Add JSON schema constraints and line-delimited prediction tracking.

Code Parallelizer

Analyze C/C++ source, convert supported Python syntax, extract loops, call classifier/generator services, validate pragmas, format output, and compile generated modules.

Benchmark Platform

Web stack for creating benchmarks and metrics, authenticating users, submitting results, and ranking pipeline performance through dashboards.
